Quick Start Guide

SharkTrack requires a terminal to run. Follow these simple steps:

Step 1: Open Terminal

Linux: Press Ctrl + Alt + T

Mac: Press Cmd + Space, type "Terminal", press Enter

Windows: Open Git Bash or WSL

Step 2: Navigate to SharkTrack Directory

cd /home/simon/Installers/sharktrack-1.5

Step 3: Launch SharkTrack Web GUI

./launch_gui.sh

Your web browser will automatically open to http://localhost:5000
